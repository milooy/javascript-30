flex-direction: 메인 축 정하기

  • flex-direction: row // 기본. 행으로 쌓임 (왼쪽 -> 오른쪽)
  • row-reverse // row가 반대로 쌓임 (오른쪽 -> 왼쪽)
  • column // 열로 쌓임(위 -> 아래)

justify-content: 메인 축 기준으로 정렬

  • justify-content: flex-start // 기본 정렬
  • flex-end // main-axis의 끝쪽으로 정렬
  • center // main-axis의 중앙으로 정렬
  • space-between // 중간에 스페이스 넣어서 정렬
  • space-around // 중간에 스페이스 넣어서 정렬하는데 처음이랑 끝도 스페이스 넣음
  • flex-direction을 column으로 했을 때는 height를 정해줘야 원하는 정렬이 나올것이다.

flex basis, grow, shrink: 기본 크기, 여백이 있을때/없을때 비율 정하기

  • flex-basis:200px; // 기본 크기는 200px
  • flex-grow: 5; // box1이 box2보다 5배 길이를 가진 것이 아니고, 200px씩을 뺀 남은 여백에서 비율이 5:1로 간다. 기본은 0이다 - 지정 안하면 여백 있어도 아무 것도 안한다
  • flex-shrink: 3; // 400px아래로 내려갔을 때 뺀 값만큼 3배로 줄어듦
  • flex: 5 3 200px; // 위를 한번에 쓴 것. grow, shrink, basis

flex-wrap: row를 감싸주기

  • flex-wrap: wrap; // 이거 없으면 그냥 한 줄에 다 나올텐데 넣으니 스르륵 내려옴(200px맞춰서)
  • flex-grow: 10; // flex-wrap으로 아래로 떨어져도, 여백의 10배 차지하는건 그 row만 기준이다. 전체 row기준 아님.
